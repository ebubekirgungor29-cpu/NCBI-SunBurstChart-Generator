# NCBI-SunBurstChart-Generator
This application allows you to easily visualize phylogenetic trees by entering organism names line by line. It automatically retrieves data from NCBI, displays interactive animations, lets you expand specific taxonomic levels, and enables saving high-resolution images.
# Installation & Usage
To run this application, first install the required dependencies by executing:
pip install -r requirements.txt

# After the installation is complete, start the application with:
python app.py

# Once the application is running, please wait until the results are displayed at the bottom of the page, as data retrieval and visualization may take some time.
